# Learn React

Let's also include some ordered list:

1. Item one
2. Item two

<Example>
  <ExampleGroup>
    <EngText>
      I <b>have</b> already <b>done</b> my homework.
    </EngText>
    <RusText>Я уже сделал домашнее задание.</RusText>
  </ExampleGroup>
  <ExampleGroup>
    <EngText>
      We have no classes today, our teacher <b>has fallen</b> ill.
    </EngText>
    <RusText>У нас сегодня не будет уроков, наш учитель заболел.</RusText>
  </ExampleGroup>
</Example>

В данных примерах глагол <b>to be</b> является самостоятельным глаголом. Так же, как и в русском языке, глагол <b>to be</b> может быть глаголом-связкой в именном сказуемом (в значении "есть").

В отличие от русского языка, в английском языке глагол-связка никогда не опускается, поскольку английское предложение имеет строго фиксированный порядок слов: подлежащее (subject) + сказуемое (verb) + дополнение (object)
